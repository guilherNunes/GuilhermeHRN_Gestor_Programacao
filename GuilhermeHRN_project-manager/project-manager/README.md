# Gestor de Projetos de Programação
## Solução Desktop para Organização de Software

---

## 🎯 Objetivo do Sistema
- Auxiliar no planejamento e monitoramento de projetos de software.
- Centralizar o controle de tarefas e prazos.
- Fornecer métricas reais de produtividade.
- Garantir a persistência e segurança dos dados.

---

## 🧩 Estrutura do Sistema
O sistema é baseado em 4 pilares principais:
1. **Usuários**: Controle de acesso por perfis (Admin, Gerente, Usuário).
2. **Projetos**: Entidade central com prazos, orçamentos e prioridades.
3. **Tarefas**: Atividades específicas vinculadas a cada projeto.
4. **Auditoria**: Histórico completo de todas as ações realizadas.

---

## 🚀 Funcionalidades Principais
- **CRUD Completo**: Gestão total de Projetos, Tarefas e Usuários.
- **Dashboard Inteligente**: Visualização de métricas de horas e progresso.
- **Sugestão de Prazo**: Algoritmo que analisa o histórico para sugerir prazos realistas.
- **Sistema de Filtros**: Busca avançada por status, prioridade e projeto.

---

## 🛠️ Requisitos Técnicos Atendidos
- **Linguagem**: Java 17 com interface gráfica Swing.
- **Arquitetura**: Organização em pacotes (Model, Service, UI, Util).
- **Persistência**: Armazenamento local em arquivos JSON.
- **Validação**: Verificação rigorosa de tipos, e-mails e campos obrigatórios.
- **Telas**: Interface composta por 5 telas funcionais integradas.

---

## 📊 Diferenciais Acadêmicos
- **Código Comentado**: Documentação didática para facilitar o entendimento.
- **Robustez**: Tratamento de erros e validações em todos os formulários.
- **Portabilidade**: Executável único (.jar) com todas as dependências incluídas.
- **Escalabilidade**: Estrutura preparada para adição de novas funcionalidades.

---

## 🏁 Conclusão
O Gestor de Projetos de Programação une a simplicidade do desktop com a robustez necessária para o gerenciamento profissional, atendendo plenamente a todos os requisitos técnicos e acadêmicos propostos.
